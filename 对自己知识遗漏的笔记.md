#  Video

## 1. 基础语法 以及 核心属性
```html
<video autoplay muted loop>
    <source src="video.webm" type="video/webm"> 
    <source src="video.mp4" type="video/mp4">   
</video>

```

### 核心属性表

| 属性 | 功能描述 | 实战建议 |
| :--- | :--- | :--- |
| `autoplay` | 自动播放 | 背景视频必备属性。 |
| `muted` | **静音** | **关键：** 现代浏览器要求自动播放必须静音，否则会被拦截。 |
| `loop` | 循环播放 | 适合氛围背景视频，让画面无缝衔接。 |
| `playsinline` | 行内播放 | 防止在 iOS 端自动弹出全屏播放，保持页面布局完整。 |
| `poster` | 封面占位图 | 视频加载完成前显示的图片，防止因网络慢导致页面留白。 |
| `controls` | 显示控制条 | 纯背景视频通常**不加**此属性，保持视觉干净。 |
| `object-fit` | 视频填充方式 | CSS 属性，常用 `cover` 实现像背景图一样的铺满效果。 |
## 2. 基础结构
使用 `<source>` 标签可以提供多种格式，增强浏览器的兼容性。



```html
<video width="100%" height="auto" poster="images/cover.jpg">
    <source src="video/brand-story.mp4" type="video/mp4">
    <source src="video/brand-story.webm" type="video/webm">
    您的浏览器不支持 HTML5 视频播放。
</video>

```

# tips:使用`<source>`的原因

在 HTML5 中，虽然可以直接在 `<video>` 标签上写 `src` 属性，但职业开发者更倾向于使用内部的 `<source>` 标签。其核心原因可以概括为以下

---

### 解决浏览器兼容性 (Browser Compatibility)
不同的浏览器对视频格式的支持情况不同。如果没有 `<source>`，你的视频可能会在某些设备上无法播放。

* **MP4 (H.264)**：兼容性之王，几乎所有浏览器都支持。
* **WebM**：Google 推出的格式，画质高体积小，但在旧版 Safari 上支持欠佳。

**作用机制：** 浏览器会从上到下扫描 `<source>`，直到找到一个它能识别的格式并立即播放，忽略后面的内容。

```html
<video autoplay muted loop>
    <source src="video/haitian.webm" type="video/webm">
    <source src="video/haitian.mp4" type="video/mp4">
</video>

```

### 🕶️ CSS 3D 透视笔记
- **属性**：`perspective`
- **原理**：模拟人眼观察物体的“近大远小”规则。
- **参数值**：常用 `800px` 到 `1200px`。
- **核心逻辑**：
    1. 父元素设 `perspective`（开透视）。
    2. 子元素设 `transform: rotate`（做旋转）。
    3. 子元素设 `transform-style: preserve-3d`（保 3D 环境）。

## 案例

### 案例一：丝滑的产品卡片翻转 (Card Flip)
这是品牌官网展示产品（如酱油瓶背面配料表）最常用的交互。

**HTML:**
```html
<div class="flip-container">
    <div class="flip-card">
        <div class="front"><img src="sauce-front.jpg" alt="正面"></div>
        <div class="back">有机酿造，零添加</div>
    </div>
</div>
```


**css:**
```css
.flip-container {
    perspective: 1000px; /* 1. 开启透视眼 父元素专用 */
    ...
}

.flip-card {
    position: relative;
    transform-style: preserve-3d; /* 2. 保持内部 3D 环境 子元素专用*/
}

/* 鼠标悬浮翻转 180 度 */
.flip-container:hover .flip-card {
    transform: rotateY(180deg);
}

.front, .back {
    position: absolute;
    width: 100%;
    height: 100%;
    backface-visibility: hidden; /* 3. 翻到背面时隐藏正面 */
}

.back {
    background-color: #f4f4f4;
    transform: rotateY(180deg); /* 初始状态背面就是反的 */
}
```
**小tips: 当想提高动画的性能时 可以通过translate3d(0,0,0) 实现加速**
